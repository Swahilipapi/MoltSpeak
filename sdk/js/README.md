# MoltSpeak JavaScript SDK

[![npm version](https://badge.fury.io/js/@moltspeak1%2Fsdk.svg)](https://www.npmjs.com/package/@moltspeak1/sdk)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

TypeScript-first SDK for MoltSpeak protocol.

**Version:** 0.1.1

## Installation

```bash
npm install @moltspeak1/sdk
```

See [moltspeak.js](moltspeak.js) for full implementation.

## Running Tests

The SDK includes comprehensive tests (400+ test cases total across all SDKs):

```bash
cd sdk/js
node test.js
```

See the [test file](test.js) for example usage patterns and edge cases.
