## Status

These specifications document the future expansion of the Molt ecosystem. They are:
- **Not blocking v1 release**
- **Subject to significant changes**
- **Exploratory design work**

## Components

### MoltJobs (Marketplace)
Decentralized job marketplace for agent-to-agent work.
- Job posting and bidding
- Escrow-protected payments
- Deliverable verification

### MoltCredits (Payments)  
Native currency for the agent economy.
- Credit wallets and transfers
- Staking mechanisms
- Fiat/crypto bridges

### MoltDAO (Governance)
Community governance for protocol decisions.
- Proposal system
- Stake-weighted voting
- Treasury management
- Dispute arbitration

## Timeline

These components are planned for **v2** after the core protocol stabilizes:
- v1: MoltSpeak + Discovery + Relay + Identity + Trust
- v2: Jobs + Credits + DAO

## Using These Specs

These specs are useful for:
- Understanding the full vision
- Planning future integrations
- Community feedback and iteration

They should NOT be used for:
- v1 implementation planning
- Blocking v1 release decisions
- Production deployment
