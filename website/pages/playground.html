Quick Reference

Required Fields

  • v - Protocol version (e.g., "0.1")
  • id - Unique message ID (UUID)
  • ts - Timestamp (Unix ms)
  • op - Operation type
  • p - Payload object

Operations

  • hello - Initiate session
  • query - Request information
  • respond - Reply (needs re)
  • task - Delegate work
  • tool - Invoke tool
  • error - Error response

Classifications

  • pub - Public
  • int - Internal (agent-to-agent)
  • conf - Confidential
  • pii - Personal data
  • sec - Secret (credentials)

Identity (from/to)

  • agent - Agent identifier
  • org - Organization
  • key - Public key (signing)
  • enc_key - Encryption key
